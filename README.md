# Nimrobo Net Studio

Web UI for the Nimrobo match network. Manage organizations, job posts, applications, and channels.

## Installation

```bash
npm install @nimrobo/net-studio
```

## Development

```bash
npm install
npm run dev      # Start development server
npm run build    # Production build
npm run start    # Start production server
npm run lint     # Run linter
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
